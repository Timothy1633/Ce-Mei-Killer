

# 🛠 微博批量拉黑工具 

这是一个可以直接在电脑浏览器控制台（Console）中运行的 JavaScript 脚本。它可以为你生成一个**可视化的悬浮操作面板**，输入目标用户的 UID，即可一键批量拉黑该用户及其所有粉丝。

尤其适用于清理恶意营销号、网络喷子（厕所账号）以及特定群体的粉丝。

## ✨ 核心功能

- 🖥️ **可视化 UI 面板**：告别盲敲代码，右下角生成深色专属操作台。
- 📜 **实时进度日志**：面板内自带微型控制台，实时滚动显示拉黑进度、成功/失败状态。
- 💥 **一键批量拉黑**：
  - **拉黑所有粉丝**：自动翻页抓取目标账号的所有粉丝并拉黑。
  - **拉黑前 5000 粉**：针对大 V 账号，抓取其活跃度最高的前 5000 名粉丝进行清理。
- ⏪ **断点续传（防封机制）**：如果因频繁操作被微博官方限制（网络请求失败），可以点击停止，休息一段时间后使用“断点继续拉黑”功能，输入上次卡住的 UID 即可接着拉黑，不用从头开始。
- ⛔ **紧急停止**：随时可以点击停止按钮中断脚本运行。

## 🚀 快速使用指南

### 第一步：打开微博网页端
在电脑浏览器（推荐使用 Chrome 或 Edge）中打开 [微博网页版 (weibo.com)](https://weibo.com) 并登录你自己的账号。

### 第二步：注入脚本代码
1. 按下键盘上的 **`F12`** 键，打开浏览器的“开发者工具”。
2. 在顶部的标签页中选择 **`Console` (控制台)**。
3. 如果控制台有警告提示（如 "Stop!"），请不要在意。将本脚本的所有代码**完整复制并粘贴**到输入框最下方，然后按下 **`Enter` (回车)** 键运行。

### 第三步：开始批量拉黑
运行成功后，网页右下角会弹出一个名为 **“🛠 微博批量拉黑工具”** 的悬浮窗口。
1. 在输入框内填入目标用户的 **微博 UID**。
2. 点击你需要的操作（如“拉黑所有粉丝”）。
3. 挂机等待即可，面板下方会实时播报拉黑进度！

## 🔍 如何获取目标用户的 UID？

**UID 是一串纯数字。** 
进入你想拉黑的人的微博主页，查看浏览器的地址栏：
- 如果地址是：`https://weibo.com/u/2303645815`，那么 UID 就是 `2303645815`。
- 如果地址是自定义域名：可以点击该用户的任意一条微博，在单条微博的 URL 中通常会包含这串数字 UID。

## ⚠️ 常见问题与注意事项

1. **为什么会提示“拉黑失败”或不断报错？（触发风控）**
   - 微博官方对 API 请求频率有严格限制（俗称“大眼制裁”）。虽然脚本中加入了 `0.5秒` 的强制延迟，但短时间内拉黑几千人依然可能触发风控。
   - **解决办法**：点击面板上的 `⛔ 停止执行`。等待 15~30 分钟后，点击 `断点继续拉黑`，输入日志中最后一个报错的 UID 即可继续。

2. **“拉黑前5000粉”没有反应 / 跨域报错？**
   - 脚本中获取前 5000 粉丝调用的是手机端接口（`m.weibo.cn`）。在纯 Console 模式下，浏览器可能会因为**CORS (跨域资源共享)** 策略拦截该请求。
   - 如果你需要完美的跨域支持及本地记忆功能，推荐将此代码转为 **Tampermonkey（篡改猴）脚本** 运行。

3. **页面刷新后进度还在吗？**
   - 当前的纯 Console 版本在**刷新页面后会丢失进度**（`window.allDogs` 会被清空）。运行期间请勿刷新网页。

## 👨‍💻 进阶：断点续传怎么用？

1. 假设你要拉黑 1000 人，在拉黑到 UID `123456789` 时报错了。
2. 点击面板上的 **断点继续拉黑**。
3. 网页会弹出一个输入框，填入 `123456789` 并确定。
4. 脚本会自动在列表中找到这个 UID，并从它开始把剩下的人拉黑。
